HotTake.gg

A full-stack, online, social media site where you can explore and share hot takes anonymously. Share your most hot takes now! Next.js is used to support the front-end with Chakra UI and Tailwind CSS contributing to the aesthetic of the social media site. The back-end utilized a Prisma schema and a Supabase database.

HotTake Mobile

A full-stack, online, social media site where you can explore and share hot takes anonymously. Now on mobile! The mobile version of HotTake utilizes React Native and Expo to support the front-end of the app, enabling touch functionality and an ease of access experience. HotTake Mobile utilizes the same Prisma and SupaBase back-end as the web-app version, accessing the same API routes, linking the mobile version and web version together!
